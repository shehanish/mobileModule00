# mobileModule00

A SwiftUI practice project for **Module 00**, containing four exercises that build up from a basic button interaction to a functional calculator UI.

## Project Structure

- `ex00`: Simple title + button with console action.
- `ex01`: Button toggles text between module title and **Hello World!**.
- `ex02`: Calculator-style UI layout with display and button grid.
- `ex03`: Improved calculator app with:
	- input handling,
	- operator replacement logic,
	- decimal-point validation,
	- expression evaluation,
	- basic error handling (invalid expression / divide by zero).

## Current Entry Point

The app currently launches `CalculatorAppView` from `mobileModule00App.swift`.

## Requirements

- Xcode (latest stable version recommended)
- iOS Simulator (used for screenshots)

## How to Run

1. Open `mobileModule00.xcodeproj` in Xcode.
2. Select an iPhone Simulator device.
3. Press **Run** (`⌘R`).
